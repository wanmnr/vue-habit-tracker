# Custom Modal Component

A flexible modal component built with Ant Design and Tailwind CSS that supports various configurations and use cases.

## Features

- Multiple variants (primary, success, warning, error)
- Different sizes (sm, md, lg, xl)
- Fullscreen toggle
- Custom header actions
- Loading state
- Configurable footer
- Responsive design

## Usage

```vue
<CustomModal
  v-model:visible="visible"
  title="Modal Title"
  variant="primary"
  size="md"
  @ok="handleOk"
  @cancel="handleCancel"
>
  <div>Modal content goes here</div>
</CustomModal>
```

Slots
The component provides several slots for customization:

default: The main content of the modal
header-actions: Additional actions in the header
footer: Custom footer content
Events
update:visible: Emitted when the modal visibility changes
ok: Emitted when the OK button is clicked
cancel: Emitted when the Cancel button is clicked
CSS Customization
The modal uses Tailwind CSS classes and can be further customized using the contentClasses prop or by extending the component styles.
